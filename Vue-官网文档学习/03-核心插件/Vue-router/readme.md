## New Words
- **route [rut] --n.路线; 航线; 路径; 线路;** 
    + an air route. 航空线路
    + The route passes close by the town. 这条路从镇子旁边经过.
    + We stopped for a picnic en route. 我们中途停下来野餐.
    + They marched the route in a day. 他们在一天内走完了全程.
    + The route is shorter than this one. 那条路比这条短.
