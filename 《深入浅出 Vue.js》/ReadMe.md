 《JavaScript 设计模式与编程实践》, 《深入浅出 Node.js》...
  这些书一样来做笔记, 主要因为此书是根据 Vue 代码中各个知识块来讲解的,
  比如 `mvvm` 便把 Vue 源码中双向数据绑定的实现原理讲一下; 虚拟 DOM
  便把什么是虚拟 DOM, Vue 中如何实现/patch/删除 这些原理讲一下, ...
  它们都不是根据源码的目录来讲解的, 而且每个章节讲完后都**没有调用示例**.
